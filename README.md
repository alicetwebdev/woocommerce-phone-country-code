# WooCommerce Phone Country Code
Add country code to phone field of WooCommerce

== Description ==

Add country code dropdown options to phone field of WooCommerce at checkout page.

== Installation ==

= Requirements =

To install WooCommerce Phone Country Code, you need:

* WordPress Version 5.5 or newer (installed)
* WooCommerce Version 3.9 or newer (installed and activated)
* PHP Version 7.4 or newer

= Instructions =

1. Log in to WordPress admin.
2. Go to **Plugins > Add New**.
3. Click on **Upload Plugin** and **Browse** to select the file `woocommerce-phone-country-code.zip`.
4. Click on **Install Now** and wait until the plugin is installed successfully.
5. Activate the plugin immediately by clicking on **Activate** on the success page; Or activate it later via **Plugins > Installed Plugins**.
